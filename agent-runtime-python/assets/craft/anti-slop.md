# Anti Slop

- No lorem ipsum or placeholder text.
- No Metric A / Card 1 / Item 1 generic labels.
- No meaningless gradient stacking.
- No bare-bones demo with only title, input, and button.
- No empty sections or placeholder cards.
- Every element on the page should serve a purpose.
